# GIS Projects

A collection of spatial analysis and GIS projects completed as part of my 
MS in Data Science (Remote Sensing) at the University of Miami.

## Projects

### [St. Lucia Terrain Analysis](./st-lucia-terrain/)
Terrain analysis of St. Lucia using SRTM 30m DEM in QGIS. 
Includes hillshade, aspect, and terrain ruggedness index (TRI) analysis.

## Tools & Technologies
- QGIS
- SRTM 30m DEM
- Remote Sensing
